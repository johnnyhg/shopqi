h1. ShoqQi

An open source ecommerce platform in Ruby on Rails
See the "demo.shopqi.com":http://www.shopqi.com:3000 . 

ShopQi was developed by "saberma":http://saberma.me

h1. Getting Started

For testing and development purposes, Vagrant can be used to bootstrap a @virtual machine@ that is ready to run ShopQi

h2. Prerequisites

h3. VirtualBox

VirtualBox 4.x is required: "http://www.virtualbox.org/wiki/Downloads":http://www.virtualbox.org/wiki/Downloads

h3. Vagrant

pre.
gem install vagrant --no-ri --no-rdoc

h3. ShopQi Box

Download the box(765M): "https://dl.dropbox.com/u/19519145/shopqi/vagrant-package-shopqi.box":https://dl.dropbox.com/u/19519145/shopqi/vagrant-package-shopqi.box
This box contains:

# Ubuntu 10.04(32bit)
# Ruby 1.9.2
# Ruby on Rails 3.0.3
# unicorn 3.0.4
# mongodb 1.6.5
# redis 2.2.0rc2
# node.js 0.2.6
# nginx 0.7.67

h2. Download the sources:

pre. 
git clone git://github.com/saberma/shopqi.git 
cd shopqi

h2. Create the VM

pre. 
#require nfsd for share folder
sudo apt-get install nfs-kernel-server
vagrant up
